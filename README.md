# Professional Portfolio Website 🚀

This repository contains my personal portfolio website, developed as an assignment for the Web Development course at **Amrita University**. The website is designed to showcase my technical skills, projects, and academic background using modern web standards.

## 🔗 Live Demo
Check out my hosted portfolio here: https://eak-47.github.io/Portfolio/

---

## 🛠️ Project Structure
The project is organized to ensure clean code and easy maintenance:
* **`index.html`**: The main entry point containing the HTML5 boilerplate and semantic structure.
* **`style.css`**: The stylesheet containing all layout, typography, and responsive design rules.
* **`/images`**: A dedicated directory for project screenshots and visual assets.

---

## ✨ Features & Requirements Met
As per the assignment criteria, this portfolio includes:
* **Navigation Menu**: A sticky header with four functional links (Home, About, Projects, Contact).
* **Project Showcase**: Detailed cards for my major projects (Hospital Management System & AI Security Researcher) including images and repository links.
* **Responsive Design**: Implemented using **CSS Flexbox** and **CSS Grid** to ensure accessibility on desktops, tablets, and mobile phones.
* **Creative Enhancements**: Integrated CSS transitions and hover animations for an interactive user experience.
* **Footer**: A footer section with relevant copyright information and quick contact details.
